## Run Locally

Clone the project

```bash
  git clone https://github.com/BassamXYZ/url-shortener.git
```

Go to the project directory

```bash
  cd url-shortener
```

Install dependencies

```bash
  pip install requirements.txt
```

Start the server

```bash
  python main.py
```
